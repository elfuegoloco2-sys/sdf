EL FUEGO LOCO — Versão estática (HTML + CSS)
=============================================
Sem backend, sem JavaScript, sem build. É só abrir o login.html no navegador.

Arquivos:
  login.html          → tela de login (entra direto no dashboard)
  dashboard.html      → painel com KPIs, ranking e zonas de atenção/crítica
  cardapio.html       → grid de pratos
  estoque.html        → tabela de itens com alertas de estoque
  relatorios.html     → gráfico SVG + ranking de descontos
  funcionarios.html   → equipe
  cozinha.html        → KDS estilo Kanban (Pendente → Preparando → Pronto → Entregue)
  garcom.html         → mapa de mesas colorido por status
  delivery.html       → cardápio + carrinho lateral
  configuracoes.html  → dados do restaurante e regras
  styles.css          → todo o tema vinho/bordô + dourado

Como abrir:
  Clique duplo em login.html — tudo navega entre si por links.

Hospedar:
  Faz upload da pasta inteira em qualquer host estático
  (Netlify Drop, GitHub Pages, Vercel, hospedagem comum).
